# Customer-Churn-Analysis - Power Bi Project

### Reducing customer churn with data-driven insights
This project analyzes a telecom customer dataset to uncover key drivers of churn—including contract type, payment method, age, and geography—and delivers actionable 
recommendations to improve retention.

### 🔍 Highlights
26.9% churn rate → $7.25M revenue lost
**Monthly contracts **= 88% of churned customers
**Top reason:** Competitor offers (45%)
**High-risk segments:** Paper Check users, older customers, CA/TX/NY residents
